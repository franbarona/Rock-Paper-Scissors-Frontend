# Rock Paper Scissors - Frontend

Angular 21 frontend for the Rock Paper Scissors web application.

## Features

- User authentication with secure JWT token handling
- Responsive game interface
- Real-time statistics and game history
- Modern UI with Tailwind CSS
- Signal-based state management
- Comprehensive test coverage
- Standalone components with dependency injection

## Tech Stack

- Angular 21
- TypeScript
- Tailwind CSS
- RxJS
- Jasmine & Karma & TestBed (Testing)

## Prerequisites

- Node.js 18+ and npm
- Angular CLI 21+
- Git


## Getting Started

### Installation

```bash
# Clone the repository
git clone 

# Navigate to frontend directory
cd frontend

# Install dependencies
npm install

# Install Angular CLI globally (if not already installed)
npm install -g @angular/cli@21
```

### Running the Application

```bash
# Start the development server
ng serve

# Navigate to http://localhost:4200/

# The app will automatically reload if you change any of the source files
```

### Running Tests

```bash
# Run all tests
ng test
```
